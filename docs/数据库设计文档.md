# 研途总控台 — 数据库设计文档

> 最后更新：2026-04-13 | 数据库引擎：SQLite (better-sqlite3, WAL 模式)

---

## ER 关系概览

```
users (1) ──< tasks (N)          users (1) ──< summaries (N)
users (1) ──< courses (N)        users (1) ──< live_sessions (N)
users (1) ──< forum_topics (N)   users (1) ──< forum_replies (N)
users (1) ──< questions (N)      users (1) ──< practice_records (N)
users (1) ──< products (N)       users (1) ──< orders (N)
users (1) ──< notifications (N)  users (1) ──< auth_tokens (N)
users (1) ──< flashcards (N)     users (1) ──< flashcard_records (N)
users (1) ──< practice_sessions (N)

tasks (1) ──< task_completions (N)
tasks (1) ──< notifications (N)
live_sessions (1) ──< live_messages (N)
forum_topics (1) ──< forum_replies (N)
questions <>─ question_tags (多对多, via question_tag_relations)
folders (1) ──< folders (N) [自引用]
folders (1) ──< folder_items (N)
flashcards (1) ──< flashcard_records (N)
```

---

## 表结构详情

### users — 用户表

| 列 | 类型 | 约束 | 说明 |
|---|---|---|---|
| id | INTEGER | PK, AUTO | 主键 |
| username | TEXT | NOT NULL, UNIQUE | 用户名 |
| password | TEXT | NOT NULL | bcrypt 哈希密码 |
| role | TEXT | NOT NULL, CHECK IN ('admin','teacher','student') | 角色 |
| display_name | TEXT | NOT NULL | 显示名称 |
| class_name | TEXT | DEFAULT '' | 班级名称 |
| openid | TEXT | DEFAULT '' | 微信 openid |
| created_at | TEXT | NOT NULL | 创建时间 ISO |

### teacher_applications — 教师注册申请

| 列 | 类型 | 约束 | 说明 |
|---|---|---|---|
| id | INTEGER | PK, AUTO | 主键 |
| username | TEXT | NOT NULL, UNIQUE | 申请用户名 |
| password | TEXT | NOT NULL | bcrypt 哈希密码 |
| display_name | TEXT | NOT NULL | 显示名称 |
| class_name | TEXT | DEFAULT '' | 班级 |
| motivation | TEXT | DEFAULT '' | 申请说明 |
| status | TEXT | CHECK IN ('pending','approved','rejected') | 状态 |
| reviewed_by | INTEGER | FK → users.id | 审核人 |
| reviewed_at | TEXT | | 审核时间 |
| created_at | TEXT | NOT NULL | 申请时间 |

### tasks — 任务表

| 列 | 类型 | 约束 | 说明 |
