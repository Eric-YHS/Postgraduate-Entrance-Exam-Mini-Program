# 更新日志

## v2.0.0 (2026-04-13)

### 新功能

- **超级管理员系统**：新增 admin 角色，教师注册审核，用户管理后台
- **题库分类**：按科目/题型/参考书分类，自定义标签系统
- **录播视频网盘**：无限层级文件夹，类似网盘的浏览体验
- **论坛增强**：支持图片/视频/附件上传和链接分享
- **艾宾浩斯词汇记忆**：SM-2 算法，四档评分，类似百词斩/不背单词
- **练习会话**：综合/按科目/错题重做模式，记录练习进度
- **学生维度视图**：教师端按学生查看任务完成情况
- **任务完成追踪**：学生可标记任务完成，教师可查看完成率
- **晚间推送提醒**：每晚 22:00 自动提醒未完成任务
- **微信订阅消息**：小程序推送通知支持
- **统一 Excel 模板**：任务/题目/用户/词汇四种标准模板

### Bug 修复

- 修复退出登录后 Bearer Token 仍有效的问题
- 修复 /api/auth/me 每次刷新新建 Token 的问题
- 修复小程序直播入口无法进入直播页的问题
- 修复 GBK CSV 导入静默失败的问题
- 修复任务允许结束时间早于开始时间的问题
- 修复正确答案可以不存在于选项中的问题
- 修复商城下单允许小数数量的问题
- 修复回复不存在的帖子返回 500 的问题
- 修复操作不存在的直播间返回成功的问题
