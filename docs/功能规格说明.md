# 研途总控台 — 功能规格说明

> 最后更新：2026-04-13

---

## 一、用户与权限

### 1.1 角色体系

| 角色 | 说明 | 默认账号 |
|------|------|----------|
| 超级管理员 (admin) | 管理教师注册审核、全部用户账号、系统统计 | admin / admin123 |
| 教师 (teacher) | 创建任务、管理题库、上传课程、直播答疑、资料商城 | teacher / 123456 |
| 学生 (student) | 查看任务、提交总结、刷题练习、词汇记忆、商城购买 | student1 / 123456 |

### 1.2 教师注册

- 公开注册页面 `/register`
- 填写用户名、密码、显示名称、班级、申请说明
- 提交后进入待审核状态
- 管理员在后台批准后自动创建教师账号

### 1.3 管理后台

- 管理员登录后跳转 `/admin`
- 功能：注册审核、用户管理（搜索/筛选/删除）、系统统计

---

## 二、任务规划

### 2.1 任务创建

- 教师 手动创建：标��、科目、开始/结束时间、周期（周几）、指定学生
- 教师 Excel 批量导入：按模板格式一键解析
- 校验：结束时间必须晚于开始时间

### 2.2 任务完成追踪

- 学生可标记任务为"已完成"
- 教师端按学生维度查看：每个学生的完成率、未完成任务列表

### 2.3 任务提醒

- 早 7:00：每日任务摘要推送（WebSocket 站内通知）
- 任务开始前 1 分钟：学习提醒
- 晚 22:00：未完成任务晚间提醒（站内 + 微信订阅消息）

---

## 三、录播视频网盘

### 3.1 文件夹层级

- 支持无限层级的文件夹嵌套（邻接表 + 递归 CTE）
- 按科目分类：如 考研数学 → 线性代数 → 真题讲解
- 类似网盘的浏览体验：面包屑导航 + 文件卡片

### 3.2 文件上传

- 支持视频文件上传和外部 URL 两种方式
- 文件信息：标题、描述、科目、文件大小

---

## 四、直播答疑

- 教师 创建直播间（draft → live → ended）
- WebRTC 实时音视频推流
- 实时文字聊天（WebSocket）
- 小程序通过 web-view 跳转，自动携带 token 认证

---

## 五、论坛社区

### 5.1 发帖与回复

- 支持分类（默认"考研交流"）
- 发帖和回复均支持：
  - 图片上传（最多 9 张）
  - 视频上传（1 个）
  - 文件附件（最多 3 个）
  - 链接分享

### 5.2 媒体渲染

- 图片：缩略图网格展示
- 视频：内嵌播放器
- 附件：下载链接
- 链接：可点击超链接

---

## 六、题库管理

### 6.1 题目分类

- **科目**：考研英语、考研数学、考研政治等
- **题型**：选择题、填空题、简答题等
- **参考书**：张宇高数18讲、肖秀荣1000题、张剑黄皮书等
- **自定义标签**：灵活的多对多标签系统

### 6.2 题目创建

- 教师 录入：标题、题干、选项 A-D、正确答案、文字解析、视频解析
- 批量导入：按 Excel 模板一键导入
- 校验：正确答案必须属于已有选项

### 6.3 学生刷题

- 选择题作答：提交后立即显示正确答案和解析
- 练习模式：综合练习、按科目练习、错题重做
- 练习会话：记录每次练习的总题数和正确率

---

## 七、词汇记忆（艾宾浩斯）

### 7.1 词汇卡片

- 教师 创建：标题、科目、正面内容、背面内容、标签
- 批量导入：按 Excel 模板导入
- 支持：正面/背面可包含文字和图片

### 7.2 复习算法

- 采用 SM-2 间隔重复算法（与 Anki 相同）
- 四档评分：Again (0)、Hard (1)、Good (2)、Easy (3)
- 自动计算下次复习日期：
  - 首次 → 1 天后
  - 第二次 → 6 天后
  - 之后 → 按难度因子递增
- 难度因子范围：1.3 ~ 无上限（默认 2.5）

### 7.3 词汇统计

- 今日待复习数量
- 已学习卡片总数
- 连续学习天数

## 八、资料商城

- 教师 创建商品：标题、描述、价格、库存、图片
- 学生下单：填写收货地址、正整数数量
- 库存自动扣减
- 订单管理

---

## 九、通知系统

### 9.1 站内通知

- WebSocket 实时推送
- 通知类型：每日任务、学习提醒、晚间提醒
- 学生端通知列表，可标记已读

### 9.2 微信推送

- 使用微信订阅消息（需用户主动订阅）
- 晚间提醒支持微信推送
- 优雅降级：未订阅用户仅收站内通知

---

## 十、数据导入导出

### 10.1 Excel 模板

| 模板 | 表头 |
|------|------|
| 任务导入 | 学生, 任务标题, 任务内容, 科目, 开始时间, 结束时间, 周期 |
| 题目导入 | 题目标题, 科目, 题型, 参考书, 题干, 选项A-D, 正确答案, 文字解析 |
| 用户导入 | 用户名, 密码, 角色, 显示名称, 班级 |
| 词汇导入 | 标题, 科目, 正面内容, 背面内容, 标签 |

### 10.2 编码兼容

- 支持 UTF-8 和 GBK 编码的 CSV/Excel
- 自动检测乱码并转码
- 导入失败时返回具体行号和错误原因

## 十一、技术规格

| 项目 | 技术栈 |
|------|--------|
| 后端 | Node.js + Express.js |
| 数据库 | SQLite (better-sqlite3, WAL 模式) |
| 前端 | 原生 HTML/CSS/JS，无构建步骤 |
| 小程序 | 微信原生小程序 |
| 实时通信 | WebSocket (ws) |
| 文件上传 | Multer |
| 定时任务 | node-cron |
| 编码转换 | iconv-lite |
| 部署 | Docker / PM2 |
