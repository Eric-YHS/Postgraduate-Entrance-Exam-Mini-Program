# 考研规划小程序任务说明

## 一、项目定位

本项目不是通用教育平台，而是一个围绕“考研规划”主题搭建的老师端 + 学生端系统。

目标是把老师给学生安排的复习任务，做成一条完整闭环：

- 规划
- 提醒
- 执行
- 复盘
- 答疑
- 刷题
- 资料转化

## 二、角色划分

### 1. 老师端

老师负责：

- 设定每日复习任务
- 按时间节点提醒学生
- 查看学生晚间总结
- 发起直播答疑
- 上传录播课程
- 维护论坛
- 上传题库
- 上架资料商品

### 2. 学生端

学生负责：

- 查看当天任务
- 接收总任务提醒与节点提醒
- 提交晚间学习总结
- 观看录播与直播
- 参与论坛交流
- 刷题
- 购买资料

## 三、当前工程形态

当前仓库已经不是单一网页，而是拆成两部分：

### Web 端

- 老师后台：保留 Web 形态
- 学生端：有 Web 演示版
- 直播间：有 Web 房间
- 后端：Express + SQLite + WebSocket + WebRTC

### 微信小程序端

- 已新增原生微信小程序学生端骨架
- 路径：`miniprogram/`
- 当前覆盖：
  - 登录
  - 今日任务
  - 提醒中心
  - 总结提交
  - 录播课程
  - 论坛
  - 商城
  - 个人中心
  - 直播 `web-view` 入口

## 四、核心功能要求

### 1. 每日复习任务

- 老师可设置每天几点到几点做什么
- 支持重复周期
- 支持按学生分配
- 支持表格导入

### 2. 自动提醒

- 每天早上 7 点推送当天总任务
- 每个任务开始时间节点再推送提醒
- 当前系统已支持：
  - 自动调度
  - 老师后台手动触发
  - Web 学生端实时接收
  - 小程序端通过接口读取提醒记录

### 3. 学习总结

- 支持文字
- 支持图片
- 支持附件
- Web 学生端可直接上传
- 小程序端当前采用“先传文字，再逐个上传图片/附件”的方式提交

### 4. 直播答疑

- 老师可创建直播间
- Web 端可本地摄像头推流
- 学生可进入直播房间
- 小程序端当前先通过 `web-view` 打开直播页

### 5. 录播课程

- 老师可上传视频文件或视频地址
- 学生端可播放
- 小程序端已接入课程页

### 6. 论坛交流

- 支持发帖
- 支持回帖
- 支持围绕考研规划、阶段复盘、薄弱科目展开交流

### 7. 题库

- 老师录题
- 支持选项题
- 支持文字解析
- 支持解析视频
- Web 学生端可直接作答
- 小程序端本轮先优先接任务、课程、论坛、商城，题库页可作为下一步继续深化

### 8. 商城

- 老师可上架资料
- 学生可模拟下单
- 老师可查看订单
- 小程序端已接商城下单入口
